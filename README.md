# groups-and-students-backend

## Cloning

`git clone git:github.com/GreatFix/groups-and-students-backend.git`

## Install

run `npm install`(or `yarn`)

## Stack
  - Nodejs
  - Express (RESTFULL)
  - PostgreSQL
    - Sequelize
